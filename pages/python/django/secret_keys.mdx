Secret Key Generator
====================

I have to create one-off Django secret keys all the time.
To do this, I use any of the following methods built-in Django methods for doing so.
This is not the only way to create a valid and secure secret key for Django, but it is what Django uses specifically.

This entire post assumes you have Django installed into a Python virtual environment.

## From the shell
```shell filename="shell_secret_key.sh" showLineNumbers copy
python -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())'
```

## From the Django Shell
```shell filename="django_shell_key.sh" showLineNumbers copy
python manage.py shell
from django.core.management.utils import get_random_secret_key
print(get_random_secret_key())
```

## Base-64 Encoded Django Secret Key
If you need a Base64 encoded version like for Kubernetes or Knative, use the following commands:
```shell filename="base_64_secret_key.sh" showLineNumbers copy
export SECRET_KEY=$(python -c "from django.core.management.utils import get_random_secret_key;print(get_random_secret_key())")
echo "Secret key is: $SECRET_KEY"
export SECRET_KEY_ENCODED=$(echo "$SECRET_KEY" | base64)
echo "Base64 Encoded Secret key is: $SECRET_KEY_ENCODED"
```
