Debian Upgrade
==============

## Upgrade Steps
1. Backup the system.
2. Update existing packages and reboot.
3. Replace the contents of `/etc/apt/sources.list` with the desired Debian version's list below.
4. Update the packages index `apt update`
5. Prepare for the operating system minimal upgrade `apt upgrade --without-new-pkgs`
6. Full upgrade the system `apt full-upgrade`

## Source Lists
### Debian 11 Bullseye
```shell
deb http://deb.debian.org/debian bullseye main contrib non-free
deb-src http://deb.debian.org/debian bullseye main contrib non-free

deb http://deb.debian.org/debian-security/ bullseye-security main contrib non-free
deb-src http://deb.debian.org/debian-security/ bullseye-security main contrib non-free

deb http://deb.debian.org/debian bullseye-updates main contrib non-free
deb-src http://deb.debian.org/debian bullseye-updates main contrib non-free

deb http://deb.debian.org/debian bullseye-backports main contrib non-free
deb-src http://deb.debian.org/debian bullseye-backports main contrib non-free
```
